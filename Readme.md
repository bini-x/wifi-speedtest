# speedtest
 
A simple CLI speed test tool written in Go using Cloudflare's endpoints.
 
> You can use this on your terminal only.
 
## Requirements
 
- [Go](https://golang.org/dl/)
 
## Usage
 
**Unix**
```bash
go build -o speedtest .
./speedtest
```
 
**Windows**
```bash
go build -o speedtest .
speedtest.exe
```
 
## Output
 
```
Testing...
Download speed: 50.20Mbps
Upload speed:   35.10Mbps
```

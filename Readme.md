# speedtest

A simple CLI speed test tool written in Go.

> You can use this on your terminal only.

## Installation

```bash
git clone https://github.com/bini-x/wifi-speedtest.git
cd wifi-speedtest
```

## Requirements

- [Go](https://go.dev/)

## Usage

**Unix**
```bash
go build -o speedtest .
./speedtest
```

**Windows**
```bash
go build -o speedtest .
speedtest.exe
```

## Output

```
Testing...
Download speed: 50.20Mbps
Upload speed:   35.10Mbps
```
